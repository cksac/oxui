# OXUI
Cross platform native GUI in Rust 

## GUI with ideas from
- Flutter 
    - Widget Tree
    - Render Object
    - Skia
- [Compose runtime](https://github.com/cksac/compose-rt)
    - State via positional memorization
    - Incremental computation

## Status
Experimental and very early stage

## Run example
- cargo run --example run_app
